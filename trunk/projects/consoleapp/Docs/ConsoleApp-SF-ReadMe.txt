Console Application Runner Classes
----------------------------------

This directory contains zip files containing Console Application Runner Classes
releases from the present release back to v1.0.2.

The source code zip files are all of the form dd-consoleapp-x.x.x.zip where
x.x.x is the version number of the release. There is also a documentation zip
file named dd-consoleapp-1.0.2-docs.zip.

Unless you have good reasons not to, you should always download the latest
version.
