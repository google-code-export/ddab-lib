Console Application Runner Classes
Demo Projects Read-Me file

The source code for ten demo projects is included with these classes, along with two console applications that are requied by the demos. All the demos are stored in subfolder of the Demos directory.

Ensure you preserve the directory structure when unzipping the downloaded archive. Open Delphi and open PJConsoleAppDemos.bpg. Compile each of the ten demo projects and the two programs in the TestApps sub-directory. All the resulting .exe files should have been written to the Exe sub-folder.

The demo programs are fully described in the Examples section of PDF user guide that can be downloaded separately. Download the user guide.
